# Sales & Revenue Analysis Dashboard

## Objective
Analyze sales performance using KPIs and visualizations.

## Dataset
Contains:
- Date
- Product
- Category
- Quantity
- Revenue

## KPIs
- Total Revenue
- Total Quantity Sold

## Visualizations
- Revenue Trend Line Chart
- Revenue by Product Bar Chart
- Revenue by Category Pie Chart

## Tools Used
- Microsoft Excel
